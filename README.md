# Projekti3-CPP-GitHub
# Sistemi për Menaxhimin e Studentëve (C++)

## Përshkrimi
Ky projekt është zhvilluar në gjuhën programuese C++ dhe ka për qëllim
demonstrimin e menaxhimit të kodit burimor dhe bashkëpunimit përmes GitHub.

## Funksionalitetet
- Shtimi i studentëve
- Shfaqja e studentëve

## Teknologjitë e përdorura
- C++
- Git
- GitHub

## Bashkëpunimi
Zhvillimi i projektit është realizuar duke përdorur:
- Branch-e për zhvillim paralel
- Commit-e të strukturuara
- Pull Request për integrimin e ndryshimeve
